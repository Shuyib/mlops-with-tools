# automatic_testing
This is an ML ops tutorial from [Elle](https://github.com/elleobrien/automatic_testing) it has been altered to suit my regular workflow.  

# Make a virtual env
```bash
make venv
```  

# Activate virtual env
```bash
source .venv/bin/activate
```  

# Install reqs
```bash
make install
```  

# Run proj
```bash
make test
```
